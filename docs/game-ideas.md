## 개발 우선순위 (추천)

### 1순위: 빠르게 만들 수 있고 검증된 재미

1. **Color Match** (스트룹 테스트)
   - 간단한 로직
   - 반응속도와 비슷한 구조
   - 심리학 기반 재미

2. **Math Speed** (빠른 암산)
   - 랜덤 수식 생성
   - 시간 제한
   - 점수 = 맞춘 개수

3. **Stack** (블록 쌓기)
   - 원탭 게임
   - 물리 불필요
   - 타이밍 게임

### 2순위: 인기 있지만 약간 복잡

4. **2048**
   - 스와이프/키보드 조작
   - 그리드 로직
   - 한글 버전 차별화 가능

5. **Pattern Copy** (패턴 따라하기)
   - 시몬 게임 스타일
   - 점점 길어지는 패턴
   - 기억력 테스트

6. **Coin Sort** (동전 정렬)
   - 2025년 트렌드
   - 머지 메카닉

### 3순위: 복잡하지만 차별화 가능

7. **Wordle 한국어**
   - 단어 DB 필요
   - 키보드 UI
   - 소셜 공유 기능

8. **Block Blast**
   - 테트리스 변형
   - Sand Tetris와 차별화 필요

## 게임 디자인 원칙 (하이퍼 캐주얼)

### 핵심 요소

1. **즉각적 이해** - 첫 탭에 규칙을 알 수 있어야 함
2. **짧은 세션** - 30초~2분
3. **중독성** - 한 판 더 하고 싶은 느낌
4. **바이럴** - 점수 공유, 도전 유발

### 피해야 할 것

- 복잡한 튜토리얼
- 긴 로딩
- 필수 회원가입
- 과도한 광고 (첫 플레이 전)
